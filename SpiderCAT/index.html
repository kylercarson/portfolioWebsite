SpiderCAT

Beckhoff Automation Intern Project

  • Designed and implemented a Spidercam-style system ("SpiderCAT") leveraging Beckhoff Automation hardware and TwinCAT, as an interactive demo for trade shows and customer events.
  • Integrated Beckhoff’s motion control libraries to coordinate a multi-axis cable-driven end effector, allowing precise movement throughout a 3D space.
  • Programmed logic to map “checkpoint” positions to represent major Beckhoff US office locations, creating a gamified user experience where checkpoints became part of an educational interactive game.
  • Implemented an Xbox controller interface where users press the A button to automatically jog the end effector to each checkpoint, using action files for rapid, modular configuration of the game's setup process.
  • Tested and tuned system safety, end effector stability, and user interface experience to ensure reliability and engagement during continuous trade show use.
  • Collaborated closely with the Beckhoff US team to align demo features with company branding and messaging.
  • Collaborated with fellow intern Orion Kraus throughout development.
